## Overview

Animeze is an app which shows information regarding Animes. Browse through the list of all the animes. The app incorporates:

- Flutter
- Jikan API
- Provider
- Sqflite
- Flutter Spinkit

Future updates will include :

- Connecting the app with a backend
- Add a Dark Mode
